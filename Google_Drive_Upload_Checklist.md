### Step 3: Upload Files
Use one of the following methods:

#### Method A: Drag & Drop (Recommended)
1. Open each local folder
2. Drag files to corresponding Google Drive folder
3. Wait for upload completion

#### Method B: Upload Dialog
1. Right-click folder in Google Drive
2. Select "Upload files" or "Upload folder"
3. Select files from local computer
4. Confirm upload

#### Method C: Google Drive Desktop
1. Install Google Drive for Desktop
2. Sync local folder to Google Drive
3. Automatic upload on sync

## Troubleshooting

### Issue: Files Won't Upload
**Solution:**
- Check file size limit (Google Drive: 5 TB per file)
- Verify internet connection
- Try uploading one file at a time
- Clear browser cache

### Issue: Sharing Link Doesn't Work
**Solution:**
- Verify sharing permissions are set
- Check recipient email address
- Try copying sharing link manually
- Ensure link isn't restricted by organization

### Issue: Files Appear Corrupted
**Solution:**
- Re-download and verify file integrity
- Re-upload problem files
- Check local file wasn't corrupted
- Contact Google Drive support if issue persists
